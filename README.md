## Installation

1. Clone the repository and enter it:

    ```sh

    ...
    $ cd bubbles/
    ```

2. Run the installation script and activate the virtual environment:

    ```sh
    $ ./scripts/install.sh
    ...

    [foobar] $ # you're good to go!
    ```

3. To check that everything is working as expected, run the tests:


    ```sh
    $ pytest tests/

    ```
